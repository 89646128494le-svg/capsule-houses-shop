# Админ-панель - Руководство

## Доступ к админ-панели

Админ-панель доступна по адресу: `/admin/login`

**Демо-доступ:**
- Email: `admin@capsule.ru`
- Пароль: `admin123`

## Функционал админ-панели

### 1. Dashboard (`/admin`)
- Общая статистика: количество заказов, выручка, новые заказы
- Последние заказы с быстрым доступом
- Карточки статистики с переходами на соответствующие разделы

### 2. Управление товарами (`/admin/products`)
- Просмотр всех товаров каталога
- Поиск по товарам
- Добавление новых товаров (UI готов, нужна интеграция с API)
- Редактирование товаров
- Удаление товаров

### 3. Управление заказами (`/admin/orders`)
- Просмотр всех заказов
- Фильтрация по статусам (новый, в обработке, отправлен, доставлен, отменен)
- Поиск по номеру заказа, имени клиента, телефону
- Изменение статуса заказа
- Удаление заказов
- Детальная информация о каждом заказе

**Статусы заказов:**
- `new` - Новый заказ
- `processing` - В обработке
- `shipped` - Отправлен
- `delivered` - Доставлен
- `cancelled` - Отменен

### 4. Аналитика (`/admin/analytics`)
- Общая выручка с динамикой
- Количество заказов
- Средний чек
- Активные клиенты
- Распределение заказов по статусам
- Топ товаров по продажам
- Место для интеграции графиков (Google Analytics / Yandex.Metrica)

### 5. Управление контентом (`/admin/content`)
- **Страницы**: Редактирование контента страниц сайта
- **Баннеры**: Управление баннерами и промо-материалами
- **Отзывы**: Модерация отзывов клиентов

## Интеграция с формами

Все формы на сайте автоматически создают заказы в админ-панели:
- Форма "Быстрый заказ" → создает заказ со статусом "new"
- Форма "Заказать звонок" → может быть интегрирована для создания лидов

## Защита роутов

Все страницы админ-панели защищены авторизацией. При попытке доступа без авторизации происходит редирект на `/admin/login`.

## Структура файлов

```
app/admin/
├── login/page.tsx          # Страница входа
├── layout.tsx              # Layout с защитой
├── page.tsx                # Dashboard
├── orders/page.tsx         # Управление заказами
├── products/page.tsx       # Управление товарами
├── analytics/page.tsx      # Аналитика
└── content/page.tsx        # Управление контентом

components/admin/
└── AdminLayout.tsx         # Layout с сайдбаром

store/
├── adminStore.ts           # Авторизация админа
└── ordersStore.ts          # Управление заказами
```
